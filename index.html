What It Finds — Real Issues Detected

From the PR #1847 audit logging module review

🔴 Critical · CWE-359 · CVSS 8.2
PII Leak in Audit Payload
src/audit/logger.ts:47
passwordHash, apiKey, and sessionToken serialized into audit_event.payload. If audit DB is compromised, all credentials are plaintext-recoverable. Fix: Apply sanitizePayload() with field-allowlist.
🔴 Critical · CWE-89 · CVSS 9.1
SQL Injection via String Interpolation
src/audit/storage.ts:83
Raw template literal: SELECT * FROM audit_log WHERE user='${userId}'. Malicious input 'admin' OR '1'='1' returns full dump. Fix: Parameterized query with $1 placeholder.
🟡 Medium · Performance
4 Closures Per Request in Hot Path
src/audit/middleware.ts:34-78
Each request creates 4 anonymous function closures. Estimated +18ms p99 under 10K RPS (SLO: +5ms). Fix: Hoist closures to module scope, pre-allocate audit context.
